# AI PDF Assistant (RAG)

An AI-powered PDF question-answering system built using Retrieval-Augmented Generation (RAG).

## Features

* Upload and query multiple PDF documents
* Semantic search using vector embeddings
* Context-aware AI responses
* Retrieval-Augmented Generation (RAG)
* ChromaDB vector database
* HuggingFace embeddings
* Groq LLM integration
* Streamlit web interface

## Tech Stack

* Python
* Streamlit
* LangChain
* ChromaDB
* HuggingFace Embeddings
* Groq API
* RAG Architecture

## Project Workflow

1. Upload PDF documents
2. Extract text from PDFs
3. Split text into chunks
4. Generate embeddings
5. Store embeddings in ChromaDB
6. Retrieve relevant context
7. Generate AI-powered answers using Groq LLM

## Example Questions

* Summarize this document
* What are the key points?
* Explain this topic in simple terms
* List all projects mentioned
* What skills are discussed?
